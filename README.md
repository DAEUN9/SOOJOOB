# 💻 기술 스택
> ### Front-End : Kotlin 1.8, Retrofit, Firebase, Google Map API
> ### Back-End : Java 1.8, Spring Boot 2.6.1, JWT, Security, JPA
> ### Server : Ubuntu 20.04, AWS, EC2, Nginx, SSL인증서
> ### DB : MySQL 8.0.28
> ### 기획 : Figma, Notion, JIRA, GitLab, ERD Cloud
